## how-tos

### create a new project from template

```bash
honey new <template-name>
```

### strict git commit

```bash
honey commit
```

### lint code

```bash
honey lint
```

### run extend command

```bash
honey extend <command>
```

### build project

```bash
honey build
```

### start development server

```bash
honey dev
```
