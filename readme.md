## how-tos

### create a new project from template

```bash
tb new <template-name>
```

### strict git commit

```bash
tb commit
```

### lint code

```bash
tb lint
```

### run extend command

```bash
tb extend <command>
```

### build project

```bash
tb build
```

### start development server

```bash
tb dev
```
